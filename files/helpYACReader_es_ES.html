Guía de comienzo rápido

YACReader es la aplicación de lectura de la suite YACReader. Puedes usarla para abrir un cómic directamente, abrir una carpeta de imágenes o continuar la lectura de algo abierto desde YACReaderLibrary.

Esta guía resume lo esencial. Para ver todos los atajos de teclado y personalizarlos, abre el diálogo de Atajos desde la barra de herramientas, el menú o el diálogo de opciones.

Empezar a leer

  • Abre un archivo de cómic, abre una carpeta o arrastra contenido a la ventana.
  • YACReader está pensado para el flujo habitual de archivos comprimidos de cómics y carpetas de imágenes. Algunas compilaciones también incluyen lectura de PDF.
  • Si abres una carpeta, YACReader puede avanzar por los archivos vecinos para que la lectura secuencial sea más cómoda.
  • La aplicación recuerda los archivos recientes y la última posición de lectura de cada cómic.

Elegir un modo de lectura

  • Usa la barra de herramientas para cambiar entre ajuste al ancho, ajuste al alto, ajuste a página completa, tamaño real y modo de desplazamiento continuo.
  • Activa el modo de doble página cuando encaje mejor con el cómic, o cambia al modo manga de doble página para invertir el orden de lectura.
  • El flujo de páginas te permite navegar visualmente antes de saltar a una página concreta.
  • La lupa está disponible durante la lectura y se puede redimensionar y ajustar de forma independiente.

Moverse por el cómic

  • Puedes leer con teclado, rueda del ratón, trackpad o con el modo de ratón seleccionado en el diálogo de opciones.
  • Usa el diálogo Ir a para saltar a una página exacta, o abre el flujo de páginas para una navegación visual rápida.
  • Los marcadores sirven para guardar páginas importantes, y YACReader también conserva la última página leída.
  • La superposición de información puede mostrar el progreso de lectura, y la herramienta de traducción se puede activar desde la barra o el menú.

Atajos

  • Abre el diálogo de Atajos para descubrir todos los atajos disponibles y cambiarlos a tu gusto.
  • El diálogo agrupa las acciones por áreas, así que es el mejor lugar para aprender qué funciones ofrece el visor.
  • Esta ayuda evita una tabla fija de atajos porque ahora son personalizables y pueden no coincidir con tu configuración.

Ajustes útiles

  • El diálogo de opciones incluye selección de idioma, apariencia y temas, color de fondo, comportamiento del flujo de páginas y controles de ajuste de imagen.
  • Puedes ajustar brillo, contraste, gamma, calidad de escalado y si las imágenes pequeñas deben ampliarse para adaptarse a la página.
  • Si usas trackpad o prefieres una forma concreta de pasar página con el ratón, revisa primero los ajustes de desplazamiento y modo de ratón.
  • Los temas claros y oscuros pueden seguir la configuración del sistema, y también se pueden cargar temas personalizados.

Si lo que quieres es gestionar una colección, buscar en ella o editar metadatos, utiliza YACReaderLibrary y abre el cómic seleccionado desde allí.
